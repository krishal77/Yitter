# Yitter

A production-grade backend platform inspired by YouTube + Twitter, built using **Node.js, Express, and MongoDB**.

---

##  Features

-  JWT Authentication system
-  Video upload system (Multer + Cloudinary)
-  Create tweets along with video upload
-  Watch history tracking
-  Likes &  comments system
-  RESTful API architecture
-  MongoDB with Mongoose schemas

---

##  Tech Stack

- Node.js
- Express.js
- MongoDB & Mongoose
- JWT (Authentication)
- Multer (File handling)
- Cloudinary (Media storage)

---

##  Key Feature

- Users can upload a **video and a tweet in a single request**, making it a hybrid of:
- YouTube (video platform)
- Twitter (text posting system)

---

## Learning Highlights

- Building real-world backend architecture
- Handling multiple media types in one API
- Designing scalable REST APIs
- Working with MongoDB relationships
- Integrating Cloudinary for media storage
- Implementing secure authentication flows
